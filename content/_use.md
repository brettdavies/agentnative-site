To audit a CLI, install the linter (`anc`) locally: see [/install](/install). To audit a website and its MCP server, use
[/web-audit](/web-audit) or the `audit_website` MCP tool. To install the agent-native-cli **skill bundle** (the Claude
Code / Codex / Cursor / OpenCode skill), see [/skill](/skill). Agents that speak MCP can connect directly to the
streamable-HTTP server at [https://anc.dev/mcp](/mcp). See [/mcp-skill](/mcp-skill) for the wire contract and tool
catalog.
