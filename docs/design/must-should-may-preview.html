Design preview for DESIGN.md §4.7. All palette tokens and keyword rules are loaded from tokens.css, which is emitted by scripts/design/generate-palette.mjs. Theme buttons toggle [data-theme]; "System" clears the override and lets prefers-color-scheme decide.

Option 7a — plain bold

P1: Non-Interactive by Default

Tools MUST run non-interactively when stdin is not a TTY. The --no-interactive flag MUST gate every prompt.

Tools SHOULD auto-detect non-interactive contexts via TTY detection (IsTerminal) and suppress prompts when stderr is not a terminal.

Tools MAY offer rich interactive experiences (spinners, progress bars, multi-select menus) when a TTY is detected.

P4: Fail Fast with Actionable Errors

MUST use try_parse() instead of parse() — the latter bypasses format-aware error output.

MUST map each error type to a distinct exit code (0, 1, 2, 77, 78 minimum).

SHOULD use a structured enum (thiserror) so agents can match on error kinds programmatically.

MAY supply a --output json escape hatch that serializes errors as JSON.
